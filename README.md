**Revenue Data and Building a Dashboard**

**Project Overview**
This project analyzes **stock price and revenue data** for companies such as **Tesla and GameStop** and visualizes the results using Python.  

It is part of the **IBM Data Science / Python for Data Science** learning labs.

**The project demonstrates:**
- Web scraping with **BeautifulSoup**
- Data extraction using **pandas**
- Data cleaning and transformation
- Visualization of stock prices and revenues

---

**Technologies Used**
- Python
- Pandas
- BeautifulSoup
- Requests
- Matplotlib
- Jupyter Notebook

---

** Files in This Repository**
Revenue Data and Building a Dashboard.ipynb`  
Main notebook containing data extraction, cleaning, and visualization

---

**Key Features**
- Scrapes quarterly revenue data for Tesla and GameStop
- Extracts historical stock prices
- Cleans revenue data by removing `$` and commas
- Converts date columns to `datetime`
- Creates clear visualizations comparing stock price and revenue

Practice web scraping
Understand financial datasets
Build dashboards using Python
Apply data cleaning techniques
